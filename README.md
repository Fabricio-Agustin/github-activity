# Activity Script python

## Descripción

Este repositorio contiene un pequeño script de automatización 
desarrollado con fines educativos y de experimentación.

Su objetivo es facilitar pruebas relacionadas con Git, GitHub y 
la automatización de tareas dentro de flujos de trabajo de desarrollo.

## Objetivos

* Experimentar con automatización mediante scripts.
* Comprender mejor el funcionamiento de Git y GitHub.
* Realizar pruebas de commits automatizados.
* Mantener un entorno de práctica para tareas repetitivas.

## Nota

La mayor parte de mis proyectos y desarrollos se encuentran en repositorios privados. Por este motivo, 
este repositorio funciona como un espacio público destinado a pruebas, aprendizaje y experimentación con 
herramientas de desarrollo y automatización.
